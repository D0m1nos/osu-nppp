osu!nppp - osu! twitch bot


format: artist - title [difficulty] (mapper) <mode>* +mods** | osu.ppy.sh/b/beatmapid | ###pp*** / 100%: ###pp, 99%: ###pp, 98%: ###pp, 95%: ###pp, 90%: ###pp
*Mode is only visible if it is not osu!standard.
**Mods are only visible if any are selected.
***Current pp, only visible if the player is currently playing a map.

so basically you click authorize and it'll bring you to a page to link the twitch account you wanna use for the bot (or just the streamer account like i like to do)
and then you just put the channel name for the channel you want to have it listen and respond to commands, then you click connect and it'll connect

also everything is client-sided so nothing is stored on a server

tosu must be running for this to work
